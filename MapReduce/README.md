# MapReduce

 - 优点： 可在短时间内完成大量工作
 - 缺点： 算法必须经过重写，需要对系统工程有一定的理解
 - 使用数据类型： 数值型和标称型数据

cat inputFile.txt | python mapper.py | python reducer.py > outputFile.txt
